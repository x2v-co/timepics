## 📌 项目简介

TimePics.ai 不是又一个 AI 图像生成器 — 它是全球首个 **AI 驱动的预测市场 + NFT 竞技场**。用户不仅能跨时间维度（过去、平行宇宙、未来）生成 AI 图像，更能将这些图像铸造为"活 NFT"，在去中心化的"时间线战争"中质押、交易、拉票、赢取真实的 SOL 奖励。

受 **Polymarket 下注机制**启发，TimePics 将 AI 艺术转化为可交易资产。想象一下：有人生成"如果罗马帝国从未灭亡"的图像并质押 NFT，另一群人生成"如果罗马更早灭亡"并对抗，社区投票决定哪条时间线获胜，获胜派系瓜分数万美元的 SOL 奖池。这不是点赞，这是真金白银的押注 — 创造了持续参与、病毒传播和多重收入流。

**核心创新：**
- 🎨 **三大时间引擎**: 回溯（修复过去）、折射（探索平行宇宙）、预见（可视化未来）
- ⚔️ **时间线战争**: Polymarket 式预测市场，NFT 作为押注仓位，社区投票决定胜负
- 🧬 **活 NFT 系统**: NFT 随时间进化（熵系统），影响战斗力和交易价值
- 🔗 **Solana Blinks**: 社交媒体一键质押和投票，病毒式传播
- 💰 **真实经济**: 质押奖励、二级市场交易、创作者收入分成

## 💡 核心功能

### 1. 🎨 三大 AI 时间引擎

**回溯引擎 (Rewind)**
- AI 超分辨率：144p → 4K 照片修复
- 时代风格迁移：1920s/1950s/1980s/2000s 美学
- 照片外延：扩展原始画面场景
- **使用场景**: "我有一张 30 年前模糊的全家福，想看清爷爷的笑容"

**折射引擎 (Refract)**
- 历史重构：生成"如果恐龙没有灭绝"的 2024 年东京
- 人脸融合：将自己植入历史事件（登月、柏林墙）
- 多重时间线：从单一事件生成 3-5 种不同走向
- **使用场景**: "如果特斯拉赢得电流战争，世界会怎样？"

**预见引擎 (Foresee)**
- 个人愿景：生成"未来的你"在梦想场景
- 年龄进展：30/50/70 岁肖像预测
- 未来快照：2050/2077/2100 年街景生成
- **使用场景**: "生成我 2050 年在火星的金婚照"

### 2. ⚔️ 时间线战争 - 核心游戏机制（Polymarket 模式）

**完整经济闭环:**

**A. 创建/加入市场**
- Pro 用户创建对战派系（如"蒸汽革命 1850" vs "生物创世纪 1850"）
- 设置战斗时长（24h/7d/30d）、最低奖池（1-100 SOL）
- 每个战斗两个对立派系争夺主导权

**B. NFT 质押 = 押注仓位**
- 生成派系主题图像 → 铸造为活 NFT (0.1 SOL)
- 质押 NFT = 你对该时间线的押注
- NFT 战力值计算：
  - 新鲜度（0-100% 熵）：新鲜 = 100 战力，古老 = 20 战力
  - 引擎加成：折射 +20%、回溯 +10%、预见 +5%
  - 社区投票：每票 +5% 战力

**C. 二级市场交易**
- 战斗中途交易质押的 NFT（投机玩法）
- 价格由获胜概率 + NFT 战力决定
- 示例：你的派系 70% 获胜概率 → NFT 价值 ↑ 40%
- 平台收取 2.5% 交易费

**D. 社交拉票战争（最关键！）**
- 每钱包每天 1 免费票（Pro 用户 5 票）
- 购买投票包：0.01 SOL = 10 票
- 分享 Solana Blink → Twitter/Discord 一键质押
- 招募队友 → 如果派系获胜，你分他们奖励的 10%
- 组织投票活动 → Discord/Telegram 协调

**E. 战斗结算与奖励**
- **获胜派系**：70% 奖池按战力分配 + "正史"徽章 + 7天0%交易费
- **失败派系**：取回 NFT + "悖论"徽章（故障艺术，收藏价值！）
- **所有选民**：20% 奖池（获胜方 2x 乘数）
- **派系创建者**：5% 所有质押（创作者收益）
- **平台抽成**：10% 奖池（开发/营销基金）

**真实案例演示:**
```
战斗："如果特斯拉赢得电流战争？"
- 派系 A（交流电）：234 NFT，1,247 票 → 28,431 战力
- 派系 B（直流电）：189 NFT，2,103 票 → 31,771 战力 ✅获胜
- 总奖池：42.3 SOL

奖励分配：
- 派系 B 质押者：29.61 SOL（平均每人 0.16 SOL，1.6x 回报）
- 所有选民：8.46 SOL（按投票数分配）
- 创建者：2.12 SOL（5% 创建费）
- 平台：4.23 SOL（10% 抽成）
```

### 3. 🧬 活 NFT 与熵系统

NFT 不是静态 JPEG，而是**随时间进化的生命体**：

- **熵值 (0-100%)**：每天自然增长 ~1-2%
  - 新鲜（0-19%）：🟢 原始质量，满战力
  - 老化（20-49%）：🟡 轻微褪色，80% 战力
  - 衰变（50-79%）：🟠 颗粒纹理，50% 战力
  - 古老（80-100%）：🔴 故障艺术，20% 战力

- **控制操作**：
  - 🔒 **冻结**：永久锁定当前熵值和战力（不可逆！）
  - ⚡ **加速**：消耗 0.01 SOL，熵值 +20%（获得稀有古老美学）

- **战略价值**：
  - 新鲜 NFT = 时间线战争高战力
  - 古老 NFT = 收藏稀缺性 + 悖论徽章美学

### 4. 🎁 每日时空胶囊

谜题游戏增加留存：
- 每 24 小时一张神秘图像 + 密码提示
- 猜对关键词解锁（失败则噪声动画）
- LocalStorage 持久化（刷新保留进度）
- 倒计时下一个胶囊

### 5. 🔗 Solana Blinks 社交病毒传播

**核心增长引擎**：
- **NFT Blinks**：Twitter 分享 NFT，粉丝一键铸造同款
- **战争 Blinks**：分享派系对战，一键质押 + 投票
- **无缝体验**：不离开 Twitter 即可完成链上操作
- **激励机制**：分享 = 拉新 = 如果获胜分 10% 奖励
