## 📋 演示内容

### 1. 首页 (Homepage)
展示了 TimePics.ai 的核心定位和功能：

#### 核心价值主张
- **Slogan**: "Render Any Moment" - 渲染任何时刻
- **Tagline**: AI驱动的视觉时间机器
- **描述**: 通过AI生成过去、平行宇宙和未来时间线的图像，并铸造为 Solana NFT

#### 三大时间引擎
1. **🔄 Rewind Engine (回溯引擎)**
   - 功能：恢复和增强老照片，回到过去
   - 特性：AI超分辨率、照片动画、时代风格迁移

2. **🔀 Refract Engine (折射引擎)**
   - 功能：可视化平行宇宙和替代历史
   - 特性：历史重建、人脸融合、"如果"场景

3. **🔮 Foresee Engine (预见引擎)**
   - 功能：生成未来愿景和年龄进展
   - 特性：未来可视化、年龄进展、趋势预测

#### 社区互动功能
- **🎁 Daily Time Capsule (每日时间胶囊)**
  - 每日时间谜题
  - 解锁隐藏的AI生成图像
  - 增强用户粘性

- **⚔️ Timeline Wars (时间线战争)**
  - 游戏化对战机制
  - 两大阵营：Steam vs Bio
  - 500 SOL 奖池
  - NFT质押统计
  - 倒计时系统
